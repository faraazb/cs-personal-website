Markerr

Markerr is a collaborative webpage annotation tool which can be used to comment on live websites. HTML elements can be tagged and text can be highlighted to discuss visual bugs or provide feedback. Markerr can eliminate feedback scattered across screenshots and multiple platforms.

Find-a-CoWIN

Find-a-CoWIN is a slot finder for MoHFW's (India) CoWIN vaccination platform. It uses the CoWIN Public API to get the vaccination slots data for any district in any state across India. It has extensive filtering options, light or dark themes and the ability to star or bookmark vaccination centers.
